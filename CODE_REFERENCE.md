Main Page Component\n**File**: `src/app/page.tsx`\n\nThis is the main UI component that coordinates the entire user experience.\n\n**Key Responsibilities**:\n- Manage application state\n- Handle user search input\n- Fetch data from API\n- Display results with sub-components\n- Handle errors and loading states\n\n**Features**:\n- Beautiful gradient header\n- Responsive layout\n- Smooth animations\n- Error messages\n- Loading indicators\n- Empty state messaging\n\n---\n\n## 2. API Route\n**File**: `src/app/api/movie/route.ts`\n\nBackend endpoint that orchestrates all data fetching and processing.\n\n**Request Flow**:\n1. Receives IMDb ID as query parameter\n2. Validates the ID format\n3. Fetches from OMDB API (movie details)\n4. Scrapes IMDb (audience reviews)\n5. Calls OpenAI API (sentiment analysis)\n6. Returns combined JSON response\n\n**Error Handling**:\n- Missing API keys → 400 error\n- Invalid requests → 400 error\n- API failures → 500 error with message\n- Network issues → Caught and reported\n\n---\n\n## 3.
